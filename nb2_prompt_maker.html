NB2 Prompt Maker ユーザーマニュアル

    
Gemini API を使って Nano Banana 2 専用プロンプトを自動生成するシングルページWebアプリです。
サーバー不要 — すべてのデータはブラウザの localStorage に保存されます。

    

      サーバー不要
      Gemini API
      Nano Banana 2 最適化
      ローカル保存
      日本語対応
    

  


  
  
📌 アプリ概要

  
NB2 Prompt Maker は「撮りたい写真のイメージ」を日本語で入力するだけで、Nano Banana 2（画像生成AI）に最適化されたプロンプトを自動生成します。

  

    
⚡
Nano Banana 2用
最適化済み英語プロンプト（200〜350字）

    
🌐
汎用英語
Flux など他の画像生成AIにも使える英語プロンプト

    
🇯🇵
日本語
内容確認・学習用の日本語プロンプト

    
🧱
JSON出力
全パラメータをJSON構造で出力。自動化に活用可能

  


  
  
🚀 クイックスタート

  
    
    
  1. Gemini APIキーを取得 — aistudio.google.com で無料取得
    2. 
    
  2. APIキーを登録 — ヘッダー右上の「🔑 API」ボタン → キー名と値を入力 → 追加
    3. 
    
  3. スタイルを選ぶ — ヘッダー下のバー：Photorealism / Anime / Cinematic / Fantasy / Product
    4. 
    
  4. プロンプトを入力 — 左パネルのテキストエリアに「撮りたい写真のイメージ」を入力
    5. 
    
  5. 「生成」ボタンを押す — または Ctrl+Enter。数秒で右パネルに結果が表示されます
    6. 
    
  6. コピーして使う — 「Nano Banana 2」パネルのコピーボタン → Nano Banana 2 にペースト
    7. 
  


  
  
🔑 APIキー管理

  
複数の Gemini API キーを登録でき、レート制限（429エラー）到達時に自動で次のキーへローテーションします。

  

    
操作方法

    
キーを追加「API」ボタン → キー名（例：Key1）＋ APIキー値を入力 → 「追加」

    
アクティブキーを切り替えキー一覧の「使用」ボタンをクリック

    
キーを削除キー一覧のゴミ箱アイコン

    
レート制限の確認ヘッダーのバー表示（0/5 件 / 60秒）でリアルタイム確認

  

  
レート上限は 5回 / 60秒 です。すべてのキーが上限に達すると生成ボタンが自動で無効になります。カウントは60秒後に自動リセット。


  
  
🎨 スタイルプリセット

  
生成するプロンプトの大きな方向性を決めます。各プリセットはAIへのシステム指示が異なります。

  

    
プリセット用途・特徴

    
Photorealismリアル写真風。スマホ撮影・CCD・フラッシュなどのカメラ描写に特化。Nano Banana 2 に最適化

    
Animeアニメ・イラスト・NovelAI / Stable Diffusion 向けスタイル

    
Cinematic映画的表現。アナモフィックレンズ・フィルムグレイン・シネマルック

    
Fantasyファンタジー・コンセプトアート。魔法・神話的景観

    
Product商品撮影・EC向け。白背景・スタジオ照明・ハイライト

  


  
  
📷 カメラ設定（Photography Bar）

  
ヘッダー下の2段目バーで撮影スタイルを細かく指定できます。設定は localStorage に自動保存されます。

  

    
設定項目説明例

    
アスペクト比生成する画像の縦横比9:16（縦）/ 16:9（横）/ 1:1

    
シーン / 背景撮影場所・環境dim bedroom interior / 夜の路上 / 桜の公園

    
カメラスタイル撮影機材・質感スマホ自撮り / CCD撮影 / boudoir撮影

    
アングルカメラ角度アイレベル / 俯瞰 / 見上げ

    
ショットタイプ画角・フレーミング全身 / 上半身 / バストアップ / クローズアップ

    
ライティング光源・照明スタイル直接フラッシュ / 窓光 / 自然光（曇り）

    
被写界深度（DOF）ピントの深さ浅い（背景ぼかし）/ 深い（全体シャープ）

  


  
  
👤 被写体プリセット

  
人物の外見情報を詳細に登録・管理できます。登録した被写体をメイン／サブに設定すると、生成時に自動的にプロンプトへ組み込まれます。

  
    
    
  1. 左パネル上部の「👤 被写体」ボタンをクリック
    2. 
    
  2. 「＋ 新規被写体」をクリック → 名前・年齢・ファッション・バイブ・顔・髪・体型・肌などを設定
    3. 
    
  3. 「メインに設定」または「サブに設定」をクリック
    メイン（琥珀色）= 主被写体、サブ（紫）= 2人目。両方設定すると2人構図プロンプトが生成されます
    4. 
  

  
設定できる項目

  

    
カテゴリ項目

    
基本名前・年齢・ファッション・バイブ（クール / チアフル / 知的 など）

    
顔顔メイク・目の形・顎のライン

    
髪シルエット・バング・カラー

    
体型フレーム・身長・バスト・ウエスト・ヒップ・肌色

    
自由記述ほくろの位置・特徴など（プロンプトに自動追加）

  


  
  
🚫 ネガティブプロンプト

  
生成プロンプトに含めたくない要素を指定します。設定は localStorage に保存され、次回も維持されます。

  

    
操作方法

    
追加テキスト入力 → Enter または「＋」ボタン

    
削除チップの「×」ボタン

    
プリセットから追加「よく使う ▼」トグル → watermark / extra fingers などのチップをクリック

  


  
  
🎨 カラーパレット

  
生成プロンプトに含めたい色をスウォッチから最大3色まで選択できます。カラートーン（暖色・寒色・モノクロ・パステルなど）も合わせて指定できます。選択した色はプロンプト内の lighting / vibe 指示に反映されます。

  
カラーパレットと生成履歴の間に MD出力ボタン（Markdownレポートのダウンロード）があります。


  
  
⚡ プロンプト生成

  
左パネル下部の「生成」ボタン（または Ctrl+Enter）でGemini APIを呼び出し、プロンプトを生成します。

  
    
    
  1. 「撮りたい写真のイメージ」を日本語または英語で入力。被写体設定・カメラ設定が自動で組み合わされます
    2. 
    
  2. 「生成」ボタンを押す。ローディング表示になり数秒で結果が右パネルに表示されます
    3. 
    
  3. レート制限（429）またはAPIエラーの場合は自動で次のキーに切り替えてリトライします
    4. 
  


  
  
📋 出力パネル

  
生成結果は右パネルに表示されます。各ブロックにコピーボタンがあります。

  

    
ブロック内容操作

    
AI解説生成プロンプトの日本語解説—

    
Nano Banana 2Nano Banana 2用の最適化済み英語プロンプト（文字数バー付き）コピー / QR / 比較 / ピン留め / Web共有

    
English Prompt汎用英語プロンプトコピー

    
Japanese Prompt日本語プロンプトコピー

    
JSON（nb_prompt.json）全パラメータをJSON構造で出力コピー

  

  
文字数バー（Nano Banana 2）

  
グリーン：0〜1200字 — 理想的な長さ

  
アンバー：1200〜1800字 — やや長い

  
レッド：1800字以上 — 長すぎ（Nano Banana 2が切り捨てる可能性あり）


  
  
📦 バッチ生成

  
同じ設定で複数のプロンプトを一度に生成します。ランダムボタンの右に「layers 3」ボタンとして表示されます。

  

    
操作方法

    
バッチ実行（3件）「layers 3」ボタンをクリック または Ctrl+Shift+B

    
件数を5件に変更バッチボタンを右クリック（3件↔5件をトグル）

    
結果の切り替え出力パネル上部のタブ（1・2・3…）をクリック

  


  
  
🌿 バリエーション生成

  
直前の生成結果をベースに、追加の指示（差分テキスト）を加えて新しいプロンプトを生成します。一度生成した後に「バリエーション」ボタンが表示されます。

  
ショートカット：Ctrl+Shift+V


  
  
🎲 ランダム生成

  
被写体・服装・シーン・ポーズなどをランダムに抽選し、確認後に生成できます。

  
    
    
  1. 「🎲」ボタンをクリック — ランダムプレビューパネルが表示され、抽選された内容が確認できます
    2. 
    
  2. 「🔄 再抽選」で別の組み合わせを試す、または「⚡ この設定で生成」で生成を実行
    3. 
  


  
  
↩️ Undo / Redo

  
生成結果をスタックで管理し、前後の出力に移動できます。入力テキストも同時に復元されます。

  

    
ショートカット動作

    
Ctrl+Z1つ前の生成結果に戻る

    
Ctrl+Y1つ先の生成結果に進む

  

  
左パネルの「Undo / Redo」ボタンでも操作できます。スタック残数が「Undo (2)」のように表示されます。


  
  
🕐 生成履歴

  
生成した結果は自動的に履歴に保存されます（最大30件）。

  

    
機能操作

    
履歴を再表示履歴一覧のアイテムをクリック

    
お気に入り登録アイテム左の ⭐ アイコン

    
絞り込みタブすべて / ⭐ / 写実 / Anime / 映画 でフィルタリング

    
検索履歴上部の検索ボックスに入力

    
履歴をエクスポート📥アイコン → JSON / CSV で書き出し

    
履歴を全削除🗑アイコン

  


  
  
📌 ピン留め

  
特に気に入った生成結果を「ピン留め」して常時アクセスできます。最大20件まで保存可能です。

  

    
操作方法

    
ピン留めNano Banana 2 パネルの「📌」ボタン（生成後に表示）

    
一覧表示ヘッダーの「📌」ボタン（件数バッジ表示）

    
削除ピン留め一覧の各アイテムのゴミ箱アイコン

  


  
  
⚖️ 並列比較ビュー

  
履歴から2件を選択して、プロンプト・スタイル・Nano Banana 2テキストを左右に並べて比較できます。

  
    
    
  1. Nano Banana 2 パネルのヘッダーにある「⚖」ボタンをクリック
    2. 
    
  2. 履歴から2件を選択（クリックでチェック。選択済みはハイライト）
    3. 
    
  3. 「比較する」ボタンで左右に並べて表示
    4. 
  


  
  
📱 QRコード

  
生成した Nano Banana 2 プロンプトをQRコードに変換して、スマートフォンで即座にスキャン・コピーできます。

  

    
操作方法

    
QR表示Nano Banana 2 パネルの「QR」ボタン

    
Web共有「共有」ボタン（Web Share API対応ブラウザのみ）

  


  
  
📑 プロンプトテンプレート

  
よく使う入力テキストをテンプレートとして保存できます（最大50件）。

  

    
操作方法

    
保存左パネルの「テンプレ保存」ボタン → テンプレート名を入力 → 保存

    
読み込み左パネルの「呼び出し」ボタン → テンプレート一覧から選択してクリック

    
削除テンプレート一覧の🗑アイコン

  


  
  
📷 設定スナップショット

  
現在のすべての設定（スタイルプリセット・モデル・アスペクト比・カメラ設定・被写体設定・ネガティブプロンプト）を1つのスナップショットとして保存・復元できます（最大30件）。

  

    
操作方法

    
保存左パネルの「スナップ」ボタン → スナップショット名を入力 → 「保存」

    
復元スナップショット一覧のアイテムをクリック

    
削除一覧の🗑アイコン

  


  
  
🎬 カメラプリセット

  
カメラ設定バーの値（スタイル・アングル・ショットタイプ・ライティング・DOF・シーン）を名前を付けて保存できます。スナップショットより軽量で、カメラ設定だけを素早く切り替えたい場合に便利です。

  

    
操作方法

    
保存カメラ設定バー右端の「🔖」ボタン → 名前を入力 → 保存

    
適用「🔖」ボタン右のプルダウンから選択

    
削除カメラプリセット一覧の🗑アイコン

  


  
  
📤 エクスポート / インポート

  
エクスポート

  

    
エクスポート対象形式操作

    
生成JSONMarkdown（.md）カラーパレット〜生成履歴の間にある「MD出力」ボタン

    
生成履歴JSON / CSV履歴パネルの「📥」アイコン

    
被写体プリセット ＋ 履歴 ＋ カメラ設定JSON（バックアップ）被写体モーダル右上の「📥」ボタン

  

  
インポート

  

    
対象操作

    
被写体 ＋ 履歴 ＋ カメラ設定被写体モーダル右上の「📤（アップロード）」ボタン → JSONファイルを選択

  

  
インポートは既存データとマージされます（重複タイムスタンプは除外）。


  
  
⭐ プロンプト品質スコア

  
生成されたプロンプトを自動採点します（0〜100点）。採点は以下の5項目に基づきます。

  

    
項目最大点評価基準

    
被写体の詳細度25点subject.description の文字数（144字で満点）

    
カメラ設定の具体性20点style / angle / shottype / lighting / depthoffield が埋まっているか

    
ネガティブプロンプト15点ネガティブ語句の数（30語で満点）

    
Nano Banana 2プロンプト長20点800字以上で満点

    
スタイル・ムード指定20点composition / style / mood フィールドの有無

  


  
  
🤖 AI改善提案

  
生成後に「AI改善提案」ボタンを押すと、AIが現在のプロンプトをレビューして改善案を提示します。

  
    
    
  1. プロンプトを生成する（事前に生成が必要です）
    2. 
    
  2. 左パネルの「AI改善提案」ボタンをクリック
    3. 
    
  3. モーダルが開き「AIが改善提案を生成中...」と表示されます
    4. 
    
  4. 完了後、改善提案・改善後プロンプト（Nano Banana 2 / 英語 / 日本語 / JSON）が表示されます
    5. 
    
  5. 「改善後テキストを入力欄に反映して再生成」ボタンで即座に適用できます
    6. 
  

  

    
操作説明

    
再生成モーダル右上の「↻ 再生成」ボタンで別の改善案を取得

    
コピー各プロンプトブロック右の「コピー」ボタンでクリップボードにコピー

    
適用して再生成「改善後テキストを入力欄に反映して再生成」ボタンで自動適用＆生成

  

  
AI改善提案は JSONスキーマなし（text/plain） でAPIを呼び出すため、通常の生成とは別枠でレート制限がかかります。


  
  
📊 統計

🖊 被写体詳細エディター

  
左パネルのカラーパレット右の「🖊」アイコンから開きます。ポーズ・服装・アクセサリー・背景などをより細かく直接記述できるフリーテキストエディターです。空欄の場合はプリセット値が使用されます。


  
  
💻 システムプロンプト編集（上級者向け）

  
ヘッダーの「</>」ボタンから各スタイルプリセットのシステムプロンプト（AIへの指示）を直接編集できます。変更は localStorage に保存され、元に戻したい場合は「リセット」ボタンを押します。

  
システムプロンプトを誤って編集すると、生成品質が大幅に低下する可能性があります。「リセット」ボタンでいつでも初期値に戻せます。


  
  
💾 下書き自動保存

  
テキストエリアへの入力は2秒ごとに自動的に localStorage へ下書き保存されます。ブラウザを閉じても次回アクセス時に「前回の下書きがあります」バーが表示され、復元できます。


  
  
🔄 リセット

  
ヘッダーの「↺」ボタンから、各種データを個別または一括でリセットできます。

  

    
リセット対象説明

    
ネガティブプロンプト登録済みNGワードをすべて削除

    
カラーパレット選択色・トーンをクリア

    
被写体詳細詳細エディターの内容をクリア

    
カメラプリセット保存済みカメラプリセットをすべて削除

    
スナップショット保存済みスナップショットをすべて削除

    
テンプレート保存済みプロンプトテンプレートをすべて削除

    
生成履歴履歴を全削除

    
統計生成統計をリセット

    
システムプロンプト編集済みシステムプロンプトを初期値に戻す

ショートカット動作

Ctrl+Shift+Bバッチ生成

    
Ctrl+ZUndo（前の生成結果へ）

    
Ctrl+YRedo（次の生成結果へ）

    
Esc開いているモーダルを閉じる

    
?ショートカット一覧を表示

  


  
  
🧠 対応モデル

  
ヘッダーのモデル選択プルダウンから使用するGeminiモデルを切り替えられます。

  

    
モデル特徴

    
gemini-2.5-flash-lite最も高速・低コスト。通常用途に最適（デフォルト推奨）

    
gemini-2.5-flash高速かつ高品質のバランス型

    
gemini-2.5-pro最高品質。複雑な被写体・詳細な指示に対応

    
gemini-2.0-flash安定版の高速モデル

    
gemini-2.0-flash-lite2.0系の最軽量モデル

  

  
品質重視なら gemini-2.5-pro、コストを抑えたい場合は gemini-2.5-flash-lite をお使いください。
